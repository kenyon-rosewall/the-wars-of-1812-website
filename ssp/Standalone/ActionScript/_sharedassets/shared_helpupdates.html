Help Update FAQ

With the release of Flash MX 2004, Macromedia will provide periodic updates to Flash Help. Here are a few of the frequently asked questions about updates.

How can I tell when a help update is available?

The Start page indicates when a new help update is available. Alternatively, you can click the Update button in the Flash MX 2004 Help panel to see if new content is available. You must be connected to the Internet for either of these functions to work.

What happens if there is more than one help update?

If you use the Help Update button, consecutive updates will be installed automatically.

Is there a PDF available of the updated content?

As each help update is released, a new PDF of each updated book will be created and posted on the Macromedia Flash support documentation page at www.macromedia.com/support/documentation/en/flash/..
