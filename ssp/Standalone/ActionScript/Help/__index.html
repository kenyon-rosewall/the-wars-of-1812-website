About the SlideShowPro for Flash component

SlideShowPro for Flash is a dynamic photo gallery and slide show component designed for maximum flexibility and customization. It is built for Flash MX 2004, Flash 8, and Flash CS3 using the v2 component architecture, and is supported by a SWF published using Flash Player 7 (or higher) and ActionScript 2.0.
