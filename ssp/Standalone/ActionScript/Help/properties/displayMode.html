SlideShowPro.displayMode

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.displayMode

Description

Property; a String that indicates the starting playback mode of SlideShowPro. This value can be "Auto" or "Manual". The default value is "Auto".

If set to Auto, SlideShowPro begins in automatic-playback mode. Images are automatically loaded one after the other using the transitionPause property to regulate how long each image stays visible before moving forward. SlideShowPro will remain in Auto mode until the user clicks on a navigational button, the play/pause button, or (if keyboard access is enabled) presses the space bar.

If set to Manual, SlideShowPro begins in manual-playback mode. The first image of the first gallery is loaded, and SlideShowPro waits for the user to click navigational buttons to change images. SlideShowPro will remain in Manual mode until the user clicks the play/pause button, or (if keyboard access is enabled) presses the space bar.

Example

The following example tells the slideshow instance to start playback in Auto mode:

my_ssp.displayMode = "Auto";
