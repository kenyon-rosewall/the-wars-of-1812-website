SlideShowPro.navLinkPreviewBackgroundAlpha

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.navLinkPreviewBackgroundAlpha

Description

Property; a Number containing the alpha transparency value for the background of the navigation link pop-up preview. The default value is 100.

Example

The following example changes the background alpha of the previews to half transparency:

my_ssp.navLinkPreviewBackgroundAlpha = 50;
