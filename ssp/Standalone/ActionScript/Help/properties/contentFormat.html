SlideShowPro.contentFormat

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.contentFormat

Description

Property; a String that assigns which viewing resolution SlideShowPro for Flash should target when loading imagery from SlideShowPro Director or from a Media RSS feed containing multiple content options within a group. Options include "Normal" (default) which uses the dimensions of SlideShowPro on the Stage of a SWF, and "Full Screen" which uses the end-user's screen resolution. "Normal" images are smaller in file size, dimension, and load faster, but are smaller when viewed full screen. "Full screen" images are larger in file size, dimension, and load slower, but are larger when viewed full screen.

Note: This property requires SlideShowPro Director and has no affect on content loaded otherwise.

Example

The following example instructs SlideShowPro for Flash to choose images appropriate for full screen viewing:

my_ssp.contentFormat = "Full Screen";
