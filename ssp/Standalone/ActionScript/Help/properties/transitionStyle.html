SlideShowPro.transitionStyle

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.transitionStyle

Description

Property; a String containing the type of transition to be applied when images unload and load in the viewer. Available options are "None", "Cross Fade", "Complete Fade", "Lens", "Photo Flash", "Wipe Top", Wipe Right", "Wipe Bottom", "Wipe Left", "Wipe Top to Background", "Wipe Right to Background", "Wipe Bottom to Background", "Wipe Left to Background". The default value is "Cross Fade".

Example

The following example sets transition style to "Complete Fade":

my_ssp.transitionStyle = "Complete Fade";
