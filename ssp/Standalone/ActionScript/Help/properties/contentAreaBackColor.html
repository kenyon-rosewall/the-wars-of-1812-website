SlideShowPro.contentAreaBackgroundColor

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.contentAreaBackgroundColor

Description

Property; a String containing the hex value for the general background behind everything in SlideShowPro. This color appears underneath all images, navigational elements, captions, and the outer stroke. The default value is "#303030".

Example

The following example changes the background color of SlideShowPro to white:

my_ssp.contentAreaBackgroundColor = "0xFFFFFF";
