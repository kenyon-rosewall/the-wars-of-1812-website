SlideShowPro.navBackgroundColor

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.navBackgroundColor

Description

Property; a String containing the hex value for the navigation bar background. The default value is "#333333".

Example

The following example changes the background color of the navigation bar to black:

my_ssp.navBackgroundColor = "#000000";
