SlideShowPro.captionBackgroundAlpha

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.captionBackgroundAlpha

Description

Property; a Number that sets the alpha (transparency) value of all photo caption backgrounds. This value can be at or between from 0 to 100. The default value is 75. Setting this to 0 effectively removes the caption background.

Example

The following example changes the alpha value of SlideShowPro's caption backgrounds to 50:

my_ssp.captionBackgroundAlpha = 50;
