SlideShowPro.audioVolume

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.audioVolume

Description

Property; a Number that sets the audio volume of sound effects, mp3s, and video. When used with mp3s and video, this is the volume level your media starts playback with. The user can then change volume on their own with the media player's volume slider thereafter if they choose to. The default value is 75.

Example

The following example tells the slideshow instance to set the playback volume all audio to 80:

my_ssp.audioVolume = 80;
