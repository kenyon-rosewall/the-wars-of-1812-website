SlideShowPro.mediaPlayerIconColor

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.mediaPlayerIconColor

Description

Property; a String containing the hex color value for the play/pause icon. The default value is light gray (#CCCCCC).

Example

The following example changes the play/pause icons to white:

my_ssp.mediaPlayerIconColor = "0xFFFFFF";
