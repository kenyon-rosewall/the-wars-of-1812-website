SlideShowPro.videoAutoStart

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.videoAutoStart

Description

Property; a Boolean that specifies whether videos auto-start when they are loaded. Options include "Off" (for no auto-start) and "On". The default is "On".

When set to "Off," SlideShowPro for Flash will look for a preview/poster graphic to display in the video area. This graphic is assigned through a "vidpreview" attribute in an img node in the XML file, and should be a complete URL to the file in relative or absolute form. The graphic should also be the same size as the video.

To add a preview graphic in the XML, modify the img node containing your video, like so:

<img src="myvideo.flv" vidpreview="gallery/album1/vid/preview.jpg" />

Or in absolute form, like so:

<img src="myvideo.flv" vidpreview="http://mydomain.com/gallery/album1/vid/preview.jpg" />

Example

The following code turns off auto-play:

my_ssp.videoAutoStart = "Off";
