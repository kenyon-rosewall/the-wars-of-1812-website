SlideShowPro.directorThumbImageSettings

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.directorThumbImageSettings

Description

Property; an Array that contains image processing settings for SlideShowPro Director to use when creating thumbnail imagery. The first index [0] represents image quality (1-100, with 100 being maximum quality), while the second index [1] represents sharpening (1-5, with 5 being the sharpest). This includes thumbnails in the "Thumbnails" navigation type, thumbnail previews, and album preview. The default values are [50,1].

Note: This property requires SlideShowPro Director and has no affect on content loaded otherwise. The sharpening option only works with servers that have ImageMagick installed.

Example

The following example tells Director to process thumbnail imagery at a quality level of 40 and with sharpening set to 5:

my_ssp.directorThumbImageSettings = [40,5];
