SlideShowPro.permalinks

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.permalinks

Description

Property; a String value for supplying permalinks. Options include "Off" and "On." If set to "On," the document path in the browser will automatically update with a direct link to the currently viewed image. If set to "Off," the document path is not modified. Note: the use of permalinks requires additional modification of the HTML document used to present SlideShowPro. See read the User Guide for more information.

Example

The following example turns permalink functionality on:

my_ssp.permalinks = "On";
