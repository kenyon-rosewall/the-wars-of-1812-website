SlideShowPro.galleryBackgroundAlpha

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.galleryBackgroundAlpha

Description

Property; a Number that sets the alpha (transparency) value of the gallery background. This value can be at or between from 0 to 100. The default value is 100. Setting this to 0 effectively removes the gallery background.

Example

The following example changes the alpha value of the gallery background to 50:

my_ssp.galleryBackgroundAlpha = 50;
