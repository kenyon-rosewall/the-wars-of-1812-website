SlideShowPro.iconInactiveAlpha

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.iconInactiveAlpha

Description

Property; a Number containing the amount of alpha transparency applied to inactive icons. The default value is 40.

During playback, SlideShowPro automatically enables and disables icons when needed. This makes the user interface clearer for end-users so they can see which icons are accessible. Setting iconInactiveAlpha to 0 effectively removes icons when they are inactive. A setting of 100 is not recommended, for there wouldn't be a visual difference between inactive and active icons.

Example

The following example makes icons invisible when they are inactive:

my_ssp.iconInactiveAlpha = 0;
