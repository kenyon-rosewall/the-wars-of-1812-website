SlideShowPro.mediaPlayerVolumeBackgroundColor

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.mediaPlayerVolumeBackgroundColor

Description

Property; a String containing the hex color value used for the background of the volume slider. The default value is black "0x000000".

Example

The following example changes the volume slider background to red:

my_ssp.mediaPlayerVolumeBackgroundColor = "0xFF0000";
