SlideShowPro.mediaPlayerTextSize

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.mediaPlayerTextSize

Description

Property; a Number containing the size of all text fields inside the media player. The default value is 9 pixels.

Example

The following example changes the size of every media player text field to 10 pixels:

my_ssp.mediaPlayerTextSize = 10;
