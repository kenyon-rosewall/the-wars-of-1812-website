SlideShowPro.navLinkPreviewAppearance

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.navLinkPreviewAppearance

Description

Property; a String that indicates whether link preview pop-ups are created when a user rolls-over an image number in the navigation bar. Options include "Hidden" and "Visible." If set to "Visible", SlideShowPro will display thumbnails if they are available in the XML file. If set to "Hidden", SlideShowPro will never display thumbnail pop-ups, even if they are available.

Example

The following example tells the instance of SlideShowPro to never show thumbnail preview pop-ups:

my_ssp.navLinkPreviewAppearance = "Hidden";
