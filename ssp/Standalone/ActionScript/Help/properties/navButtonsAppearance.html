SlideShowPro.navButtonsAppearance

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.navButtonsAppearance

Description

Property; a String that indicates which controls / buttons should be made available in the navigation. Options include "All Visible", "Hidden", "Hide Display Mode Control", "Hide Gallery Control", "Hide Full Screen Control", "Hide Display Mode and Gallery Control", "Hide Display Mode and Full Screen Control", "Hide Display Mode and Gallery and Full Screen Control", "Hide Gallery and Full Screen Control". The default value is "All Visible". Setting this to "Hidden" will remove all controls from the navigation and leave only the navigation links.

Example

The following example tells the slideshow instance to hide the play/pause buttons:

my_ssp.navButtonsAppearance = "Hide Display Mode Control";
