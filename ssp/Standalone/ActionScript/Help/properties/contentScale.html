SlideShowPro.contentScale

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.contentScale

Description

Property; an Array that indicates how SlideShowPro scales slideshow content. This value can be "None", "Crop to Fit", "Crop to Fit All", "Proportional", or "Downscale Only". The default value is "Downscale Only".

If set to "None," content is loaded without any scaling applied. If set to "Crop to Fit", content will proportionally downscale or upscale to completely fill the viewing area of the component, with excess area cropped out. If set to "Crop to Fit All", content scales/crops the same way as "Crop to Fit," but the entire area of the component is used. If set to "Proportional," content will proportionally downscale or upscale to fit the dimensions of the component. If set to "Downscale Only," content will proportionally downscale if larger than the component, but will not be upscaled if smaller.

Example

The following example tells the instance of SlideShowPro to always proportionally scale slideshow content:

my_ssp.contentScale = "Proportional";
