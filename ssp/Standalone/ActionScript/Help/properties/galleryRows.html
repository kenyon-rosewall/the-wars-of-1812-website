SlideShowPro.galleryRows

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.galleryRows

Description

Property; a Number that indicates the maximum number of rows in each gallery screen. The default value is 4.

If value creates more rows than the gallery screen can display SlideShow will automatically decrement this value until reaching the maximum number that will fit. If the opposite is true and there is more vertical gallery space than number of assigned rows, each row will expand vertically to fill the available space.

Note: The minimum size of an album element is 160px wide by 66px high when "Album Type" is set to "Thumb Crop to Fit." If "Thumb and Text" is used, the minimum size is 25px wide by 25px high.

Example

The following example tells the slideshow instance create five rows in each gallery screen to contain gallery items:

my_ssp.galleryRows = 5;
