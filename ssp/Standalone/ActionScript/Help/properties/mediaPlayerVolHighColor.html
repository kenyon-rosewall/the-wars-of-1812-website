SlideShowPro.mediaPlayerVolumeHighlightColor

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.mediaPlayerVolumeHighlightColor

Description

Property; a String containing the hex color value used for the front of the volume slider. The default value is light grey "0xCCCCCC".

Example

The following example changes the volume slider background to red:

my_ssp.mediaPlayerVolumeHighlightColor = "0xFF0000";
