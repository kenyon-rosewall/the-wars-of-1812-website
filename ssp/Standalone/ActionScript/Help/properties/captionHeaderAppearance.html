SlideShowPro.captionHeaderAppearance

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.captionHeaderAppearance

Description

Property; a String containing the type of caption header to be used. Options include "Hidden," "Image Count", "Image Title", "Album Title", and "File Name". The default value is "Image Count".

  • "Hidden" -- removes the caption header.
  • "Image Count" -- will display "Image [image number] of [total album images]". The English words "Image" and "of" are changeable through the "Text String" parameter.
  • "Image Title" -- the title of an image as assigned in the XML file by adding a "title" attribute to an image element.
  • "Album Title" -- the title of the album from which the current image originates.
  • "File Name" -- the file name of the image. This is equal to the "src" attribute value in an image element in the XML file.

Example

The following example changes the caption header type to "Album Title":

my_ssp.captionHeaderAppearance = "Album Title";
