SlideShowPro.navThumbLinkShadowAlpha

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.navThumbLinkShadowAlpha

Description

Property; a Number containing the level of transparency to apply to shadows underneath thumbnail navigation links. Acceptable range is from 0 (completely transparent) to 100 (opaque). The default setting is 60.

Example

The following example removes the shadows underneath thumbnail links in the navigation:

my_ssp.navThumbLinkShadowAlpha = 0;
