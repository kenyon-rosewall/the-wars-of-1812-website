SlideShowPro.contentFrameStrokeColor

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.contentFrameStrokeColor

Description

Property; a String containing the hex color value for strokes applied around slideshow content backgrounds. The default value is "#333333".

Note: In order to see content strokes, Content Frame Stroke Appearance must be set to "Visible"

Example

The following example changes the image stroke color to white:

my_ssp.contentFrameStrokeColor = "0xFFFFFF";
