SlideShowPro.albumStrokeColor

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.albumStrokeColor

Description

Property; a String containing the hex value for all album outlines. The default value is "#333333".

Example

The following example changes the color of all album outlines to black:

my_ssp.albumStrokeColor = "0x000000";
