SlideShowPro.soundEffects

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.soundEffects

Description

Property; an Array containing the linkage variables of the sound clips in the FLA Library to be used for interface sound effects.

soundEffects[0] designates a sound to use when a navigation link, button, or album is clicked. soundEffects[1] designates a sound to use when a navigation link or album is rolled-over. Finally, soundEffects[2] designates a sound to use when the previous link group button, the next link group button, the previous screen button, or the next screen buttons are clicked.

Example

The following example sets the sound clip with a linkage variable of "clickSound" to use for link clicks:

my_ssp.soundEffects[0] = "clickSound";
