SlideShowPro.navLinkRolloverColor

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.navLinkRolloverColor

Description

Property; a String containing the hex color value of navigation links when rolled over. When using the "Numbers" Navigation Link Appearance, the text is restyled with this color. When using "Thumbnails", the stroke around the thumbnail image is restyled with this color. The default value is "#FFFFFF".

Example

The following example changes the navLinkRolloverColor hex value to red:

my_ssp.navLinkRolloverColor = "0xFF0000";
