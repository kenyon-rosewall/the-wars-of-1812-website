SlideShowPro.albumTitleSize

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.albumTitleSize

Description

Property; a Number containing the point size for the title text field of each album. The default value is 9.

Example

The following example changes the point size of each album title text field to 8:

my_ssp.albumTitleSize = 8;
