SlideShowPro.contentAreaStrokeColor

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.contentAreaStrokeColor

Description

Property; a String containing the hex value for the outer stroke around SlideShowPro. This stroke appears above all other content. The default value is "#262626".

Example

The following example changes the outer stroke color of SlideShowPro to black:

my_ssp.contentAreaStrokeColor = "0x000000";
