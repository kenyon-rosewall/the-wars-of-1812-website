SlideShowPro.mediaPlayerAppearance

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.mediaPlayerAppearance

Description

Property; a String that controls whether the media player interface is displayed when audio or video content is playing back. Options include "Hidden", "Always Visible" and "Visible". The default value is "Visible." If set to "Hidden," the media player never appears. If set to "Always Visible," the player is constantly visible when audio/video playback is occurring. If set to "Visible," the player is visible at the beginning of audio/video playback, and after 5 seconds automatically disappears. It then reappears when the mouse pointer is near where the player should normally be.

Example

The following example instructs the slideshow instance to never show the media player:

my_ssp.mediaPlayerAppearance = "Hidden";
