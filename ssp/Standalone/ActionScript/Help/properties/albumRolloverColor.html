SlideShowPro.albumRolloverColor

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.albumRolloverColor

Description

Property; a String containing the hex value an album's background changes to when the user's mouse rolls-over the item. Album background returns to albumBackgroundColor on roll-out. This value is also used by the "Previous" and "Next" screen buttons in the Gallery window. The default value is "#262626".

Example

The following example changes the roll-over background color of each album to white:

my_ssp.albumRolloverColor = "0xFFFFFF";
