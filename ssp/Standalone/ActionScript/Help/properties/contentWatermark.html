SlideShowPro.contentWatermark

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.contentWatermark

Description

Property; a String containing linkage ID of the movie clip in the Flash Library panel to use as the watermark image.

Example

The following code assigns a movie clip with a name of "waterClip_mc" to SlideShowPro:

my_ssp.contentWatermark=waterClip_mc;
