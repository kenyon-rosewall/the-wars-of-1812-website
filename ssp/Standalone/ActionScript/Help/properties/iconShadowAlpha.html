SlideShowPro.iconShadowAlpha

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.iconShadowAlpha

Description

Property; a Number containing the amount of shadow -- both overlay and underlay -- applied to each icon (when the default icons are used). The default value is 60. Setting this to 0 removes the shadow effect all-together.

Example

The following example lightens the shadow effect:

my_ssp.iconShadowAlpha = 40;
