SlideShowPro.galleryOrder

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.galleryOrder

Description

Property; a String that indicates the order in which gallery items are positioned in each screen. This value can be "Left to Right" or "Top to Bottom". The default value is "Left to Right."

When "Left to Right" is chosen, the first gallery item (as stipulated by the top-down sequence of the XML) is placed in the top-left corner of the gallery screen. Subsequent items are then placed to the right of the first item until reaching the maximum amount of columns allowed. Items then shift to the next row, starting again from the left.

When "Top to Bottom" is chosen, the first gallery item (again stipulated by the XML) is placed in the top-left corner of the gallery screen. Subsequent items are then placed below the first until reaching the maximum number of rows. Items then shift to the next column, starting again from the top.

Example

The following example tells the slideshow instance to place gallery items in a left-to-right order:

my_ssp.galleryOrder = "Left to Right";
