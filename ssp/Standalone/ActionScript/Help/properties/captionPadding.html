SlideShowPro.captionPadding

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.captionPadding

Description

Property; an Array that dictates the amount of padding to apply around text fields in the caption area. Like CSS, the array assigns padding values in the following order: top, right, bottom, left. The default value is 5 for all four values.

Example

The following example tells the slideshow instance to place 10 pixels of padding on top and below (but keeps the default 5 for left and right) each text field in the caption area:

my_ssp.captionPadding = [10,5,10,5];
