SlideShowPro.videoBufferTime

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.videoBufferTime

Description

Property; a number that specifies the number of seconds to buffer in memory before beginning to play a video. For a progressively downloaded FLV file over HTTP, there is little benefit to increasing this value although it could improve viewing a high-quality video on an older, slower computer.

NOTE: This property does not specify the amount of the FLV file to download before starting playback.

Example

The following code sets SlideShowPro's video buffer time to 4 seconds:

my_ssp.videoBufferTime = 4;
