SlideShowPro.autoFinishMode

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.autoFinishMode

Description

Property; a String that indicates how SlideShowPro behaves when reaching the final image in an album. If set to "Switch," auto-play is maintained, and the next album is loaded. If set to "Stop," auto-play changes to manual, and the slide show stops. If set to "Restart," auto-play is maintained, and the current album is restarted. If set to "Open Gallery," the gallery interface is opened. This mode functions with both Random and Sequential playback. Note: this property replaces the "Auto Switch Albums" property. The default value is "Switch."

Example

The following example tells the slideshow instance to stop auto-playback at the final image of the current album:

my_ssp.autoFinishMode = "Stop";
