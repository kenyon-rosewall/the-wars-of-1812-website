SlideShowPro.contentWatermarkAlign

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.contentWatermarkAlign

Description

Property; a String containing the position of the attached watermark clip ("watermark") in relation to each slideshow content asset. Options include "Center", "Top Left", "Top Center", "Top Right", "Center Right", "Bottom Right", "Bottom Center", "Bottom Left", "Center Left". The default value is "Bottom Right".

Example

The following code aligns watermarks to the top left of each slideshow content asset:

my_ssp.contentWatermarkAlign="Top Left";
