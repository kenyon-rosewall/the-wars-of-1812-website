SlideShowPro.galleryAppearance

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.galleryAppearance

Description

Property; a String that indicates how the gallery screen interface is included with SlideShowPro when published. This value may be "Hidden", "Closed at Startup" or "Open at Startup". "Hidden" removes the gallery interface all together, including the button icon in the navigation bar. "Closed at Startup" includes the gallery, starts your slideshow with the gallery closed and auto-loads the first album. "Open at Startup" includes the gallery as well, but is displayed when the component is initially loaded so that the user may pick which album they want to view.

Example

The following example tells the slideshow instance to not include the gallery interface:

my_ssp.galleryAppearance = "Hidden";
