SlideShowPro.cacheContent

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.cacheContent

Description

Property; a String that indicates whether SlideShowPro should cache (in advance) images and video from albums when loaded. Options include "None", "Only Images", "Only Thumbnails", and "All". If set to "None", no content is cached and everything is loaded on-demand. If set to "Only Thumbnails", only thumbnail imagery for the navigation and link previews are cached in-advance, and all slideshow content is loaded on-demand. If set to "Only Images," only thumbnail imagery and slideshow images (no videos) are cached in advance. If set to "All", thumbnails, slideshow images, and slideshow videos are all cached in advance. The default value is "None".

Example

This example instructs SlideShowPro to preload all thumbnail images for the current album:

my_ssp.cacheContent = "Thumbnails";
