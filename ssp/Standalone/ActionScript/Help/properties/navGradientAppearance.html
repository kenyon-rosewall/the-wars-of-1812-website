SlideShowPro.navGradientAppearance

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.navGradientAppearance

Description

Property; a String that sets the appearance of the gradient overlay that appears on top of the navigation background. Options include "Hidden", "Glass Dark", "Glass Light", "Smooth Dark", "Smooth Light". The default value is "Glass Dark".

Example

The following example changes the removes the gradient overlay:

my_ssp.navGradientAppearance = "Hidden";
