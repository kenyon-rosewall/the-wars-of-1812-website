SlideShowPro.contentOrder

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.contentOrder

Description

Property; a String that indicates the order in which slideshow content is loaded and displayed in SlideShowPro. This value can be "Random" or "Sequential". The default value is "Sequential".

If set to "Sequential," content is loaded in numerical order (the order in which they are placed in the xml). If set to "Random," content is loaded in true random order -- meaning you won't see the same image twice until all content from an album has loaded.

Example

The following example tells the slideshow instance to load content in random order:

my_ssp.contentOrder = "Random";
