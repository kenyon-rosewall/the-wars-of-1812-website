SlideShowPro.typeface

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.typeface

Description

Property; a String containing one or more typeface names to be used as the general font for SlideShowPro. The default value is "Lucida Grande,Lucida Sans Unicode,Verdana,Arial,_sans". Note: do not include spaces between commas and a font name.

When using "System" fonts -- typefaces on nearly every user computer -- this value may be a comma-separated list of typeface names. SlideShowPro will use the fist available typeface in the comma-separated list.

When using embedded typefaces, enter the Linkage identifier of your typeface in the library. Note: be sure to set the value of typefaceEmbed to true or else your text will not appear.

Example

The following example sets the System font "Georgia" or any serif as SlideShowPro's general typeface:

my_ssp.typeface = "Georgia, serif";

The following example sets the embedded typeface "standard" as SlideShowPro's general typeface:

my_ssp.typeface = "standard";
