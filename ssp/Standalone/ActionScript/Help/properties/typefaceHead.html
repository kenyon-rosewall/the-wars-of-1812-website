SlideShowPro.typefaceHead

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.typefaceHead

Description

Property; a String containing one or more typeface names to be used as the "header" font for SlideShowPro. This applies to gallery item titles and photo caption titles. The default value is "Lucida Grande,Lucida Sans Unicode,Verdana,Arial,_sans".

When using "System" fonts -- typefaces on nearly every user computer -- this value may be a comma-separated list of typeface names. SlideShowPro will use the fist available typeface in the comma-separated list.

When using embedded typefaces, enter the Linkage identifier of your typeface in the library. Note: be sure to set the value of typefaceEmbed to true or else your text will not appear.

Example

The following example sets the System font "Arial" or any serif as SlideShowPro's header typeface:

my_ssp.typefaceHead = "Arial";

The following example sets the embedded typeface "standardBold" as SlideShowPro's header typeface:

my_ssp.typefaceHead = "standardBold";
