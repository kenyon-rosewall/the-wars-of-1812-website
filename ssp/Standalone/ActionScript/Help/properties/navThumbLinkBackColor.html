SlideShowPro.navThumbLinkBackgroundColor

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.navThumbLinkBackgroundColor

Description

Property; a String containing the hex value for the background of the thumbnail pop-up preview and the thumbnails in the navigation bar. The default value is "#666666".

Example

The following example changes the background color of the navigation area's thumbnails to black:

my_ssp.navThumbLinkBackgroundColor = "0x000000";
