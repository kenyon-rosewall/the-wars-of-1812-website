SlideShowPro.albumBackgroundAlpha

Availability

Flash Player 7

Edition

Flash MX 2004.

Usage

slideShowProInstance.albumBackgroundAlpha

Description

Property; a Number that sets the alpha (transparency) value of each album background in the gallery. This value can be at or between from 0 to 100. The default value is 100. Setting this to 0 effectively removes each album's background.

Example

The following example changes the alpha value of each album background to 50:

my_ssp.albumBackgroundAlpha = 50;
